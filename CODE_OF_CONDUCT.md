# Code of Conduct

Codex Cortex follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project respectful and focused on useful, safe collaboration.

## Expected Behavior

- Be respectful and constructive.
- Keep discussion technical and project-relevant.
- Report safety or security concerns clearly.
- Do not share credentials, private data, or private chain-of-thought.

## Unacceptable Behavior

- Harassment or abuse.
- Sharing private information without permission.
- Attempts to bypass approval gates or hide risky behavior.
- Submitting code or documentation that intentionally misleads users about what the project does.

## Enforcement

Maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct.
