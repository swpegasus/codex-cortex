# Cortex Closeout

Use this prompt before ending a meaningful work session.

Update:

- `.cortex/STATE.md`
- `.cortex/TASKS.md`
- `.cortex/SESSIONS.md`
- `.cortex/WORKLOG.md`
- `.cortex/LEARNINGS.md`, if durable lessons were found
- `.cortex/RECOVERY.md`, if safe resume steps changed

Final response should say what changed, what was verified, what was not verified, and what remains blocked.
